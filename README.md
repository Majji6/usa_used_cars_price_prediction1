# 🚗 USA Used Cars Price Prediction
## Overview
This project predicts the selling price of used cars in the USA using Machine Learning.

## Features
- Data preprocessing
- Feature engineering
- Machine Learning model
- Gradio web application
- Deployed on Hugging Face

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Gradio

## Live Demo
https://huggingface.co/spaces/MAJJI12/usa_cars

## Dataset
USA Used Cars Dataset
